# Password strength checker

Overview:
The command "psc" provides a password strength assessment functionality.
This application accepts a password string as an input and returns an evaluation of its strength based on a set of predefined criteria.
The maximum length of the password is 100 characters.

The evaluation criteria for "Very Weak", "Weak", "Medium", "Strong", and "Very Strong" are defined as follows:

- "Very Weak": Password has less than 8 characters or contains only one type of characters (either only lowercase, only uppercase, only digits, or only special characters).
- "Weak": Password is at least 8 characters long and includes at least two types of characters (e.g., lowercase and digits, uppercase and special characters, etc.)
- "Medium": Password is at least 10 characters long and includes at least two types of characters, one of which is a special character.
- "Strong": Password is at least 12 characters long and includes at least three types of characters, one of which is a special character.
- "Very Strong": Password is at least 16 characters long and includes all types of characters - lowercase, uppercase, digits, and special characters.

The password strength levels are determined from the weakest to the strongest in a top to bottom order.
Special characters include symbols `` !"#$%&'()*+,-./:;<=>?@[]^_`{|}~ ``

Example input 1:
`psc myPassword123`

Example output 1:
`The provided password is "Weak" strength.`

Example input 2:
`psc ComplexP@ssword#123`

Example output 2:
`The provided password is "Very Strong" strength.`
